# 小红书笔记获取插件

这是一个用于Dify平台的小红书笔记获取插件。它可以通过笔记ID或分享链接获取小红书笔记的详细内容。

## 功能

- 通过笔记ID获取小红书笔记内容
- 通过分享链接获取小红书笔记内容
- 返回包含笔记详情、作者信息、统计数据等内容

## 使用方法

函数签名：`get_xiaohongshu_note(note_id=None, share_url=None)`

参数：
- `note_id`: 小红书笔记ID（可选）
- `share_url`: 小红书笔记分享链接（可选）

注意：必须提供`note_id`或`share_url`中的至少一个参数。
